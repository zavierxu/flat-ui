组件

按钮

经过重新设计的 Bootstrap 按钮, 我们决定增加一个新的尺寸：btn-hg 被用于 这个页面最主要的动作： 

<button class="btn btn-hg btn-primary">
  超级大按钮
</button>

此外，我们增加了 btn-embossed 使得按钮看起来更加真实，但是我们更喜欢扁平。 

<button class="btn btn-embossed btn-primary">
  浮雕按钮
</button>

在某些情况下按钮需要更宽。是的，我们已经注意到了，就象这样： 

<button class="btn btn-default btn-wide">
  取消
</button>

另一个新的东西是 btn-tip. 当你需要强调按钮文本某一部分的时候，请使用它： 

<button class="btn btn-primary">
  保存
  <span class="btn-tip">￥300</span>
</button>

你可以照常设置不同的颜色： 

<button class="btn btn-default">默认</button>
<button class="btn btn-primary">主要</button>
<button class="btn btn-info">信息</button>
<button class="btn btn-danger">危险</button>
<button class="btn btn-success">成功</button>
<button class="btn btn-warning">警告</button>
<button class="btn btn-inverse">反色</button>

按钮组、按钮工具栏、禁用状态也可以照常使用。

复选框

使用扁平风格的复选框需要JS插件为我们创建自定义标记，我们来看一下：

数据属性：
<label class="checkbox" for="checkbox1">
  <input type="checkbox" value="" id="checkbox1" data-toggle="checkbox">
  Checkbox
</label>
通过JS调用:
$(':checkbox').checkbox();
方法：

.checkbox('toggle')
来回切换 选中 和 未选中 状态。

.checkbox('check')
设置复选框状态为 选中。

.checkbox('uncheck')
设置复选框状态为 未选中。 

$(':checkbox').checkbox('check');
事件：

.on('toggle')
当复选框状态在 选中 和 未选中之间变化时终止。

.on('change')
和 toggle相同。 

$(':checkbox').on('toggle', function() {
  // Do something
});

单选按钮

重设计后的单选按钮也需要JS插件，我们尝试着将事情变得简单，所以你也不用费力和自定义组件 打交道。

数据属性：
<label class="radio">
  <input type="radio" name="group1" value="1" data-toggle="radio">
  单选框未选中
</label>

<label class="radio">
  <input type="radio" name="group1" value="2" data-toggle="radio" checked>
  单选框选中
</label>
JS调用：
$(':radio').radio();
方法：

.radio('check')
将单选按钮状态设置为 选中.

.radio('uncheck')
将单选框状态设置为未选中. 

$(':radio').radio('check');
事件：

.on('toggle')
当单选按钮的状态在 选中 和 未选中.

之间切换时fired。

.on('change')
和toggle 相同，只是 — 当各单选框有状态变化时才会fire。 

$(':radio').on('toggle', function() {
  // Do something
});

选择

由于无法重新设计系统下拉菜单，我们同时做了自定义下拉菜单。它基于 Selectpicker 插件， 做了小小的改动来适配Bootstrap的命名规则。

这个组件继承了整个按钮的样式，所以你可以自由使用所有的颜色和尺寸。我们同样支持 多选 状态以及block 模式，可以利用所有的可用空间。

网格也同样适用。更多信息请阅读 原始文档。

你可以和往常一样使用标记： 

<select>
  <option value="0">我的概述</option>
  <option value="1">我的朋友</option>
</select>

<select multiple="multiple">
  <option value="0">我的概述</option>
  <option value="1" selected>我的朋友</option>
  <option value="2" selected>我的书籍</option>
  <option value="3">My CDs</option>
</select>

然后使用JS来把系统下拉菜单改为自定义下拉菜单。 

$("select").selectpicker({style: 'btn-hg btn-primary', menuStyle: 'dropdown-inverse'});

// style: select toggle class name (which is .btn)
// menuStyle: dropdown class name

// You can always select by any other attribute, not just tag name.
// Also you can leave selectpicker arguments blank to apply defaults.

滑块

重新设计的jQuery UI slider 增加了分块滑动。

标记：

<div id="slider" class="ui-slider"></div>

JS:

var $slider = $("#slider");
if ($slider.length > 0) {
  $slider.slider({
    min: 1,
    max: 5,
    value: 3,
    orientation: "horizontal",
    range: "min"
  }).addSliderSegments($slider.slider("option").max);
}

addSliderSegments()如何工作？

这是一个扩展的jQuery方法，放置在application.js文档里。通过slider("option").max（显示最大值）来计算需要分多少段。

// Add segments to a slider
$.fn.addSliderSegments = function (amount, orientation) {    
  return this.each(function () {
    if (orientation == "vertical") {
      var output = ''
        , i;
      for (i = 1; i <= amount - 2; i++) {
        output += '<div class="ui-slider-segment" style="top:' + 100 / (amount - 1) * i + '%;"></div>';
      };
      $(this).prepend(output);
    } else {
      var segmentGap = 100 / (amount - 1) + "%"
        , segment = '<div class="ui-slider-segment" style="margin-left: ' + segmentGap + ';"></div>';
      $(this).prepend(segment.repeat(amount - 2));
    }
  });
};

自定义滑块数值：

标记：

<div id="slider3">
  <span class="ui-slider-value first"></span>
  <span class="ui-slider-value last"></span>
</div>

JS:

var $slider3 = $("#slider3")
  , slider3ValueMultiplier = 100
  , slider3Options;

if ($slider3.length > 0) {
  $slider3.slider({
    min: 1,
    max: 5,
    values: [3, 4],
    orientation: "horizontal",
    range: true,
    slide: function(event, ui) {
      $slider3.find(".ui-slider-value:first")
        .text("$" + ui.values[0] * slider3ValueMultiplier)
        .end()
        .find(".ui-slider-value:last")
        .text("$" + ui.values[1] * slider3ValueMultiplier);
    }
  });

  slider3Options = $slider3.slider("option");
  $slider3.addSliderSegments(slider3Options.max)
    .find(".ui-slider-value:first")
    .text("$" + slider3Options.values[0] * slider3ValueMultiplier)
    .end()
    .find(".ui-slider-value:last")
    .text("$" + slider3Options.values[1] * slider3ValueMultiplier);
}
